# waifudex

**My codex is waifu!**

[Codex CLI](https://github.com/openai/codex) status monitoring tool with kawaii character.

## Why do I have to use this?

- You can easily check the progress of Codex's work without having to stare at the terminal.

- Having an anime character on the screen makes you look like a skilled developer. (Seriously)

- **Kawaiiiiiii**

## Features

- Codex CLI Live Monitoring
- Kawaii Agent in my Desktop (based on [Inochi2D](https://github.com/inochi2d))
